# DNS (Domain Name System)

## What is DNS?

DNS stands for Domain Name System.

It translates domain names into IP addresses.

Example:

google.com → 142.250.x.x

Instead of remembering IP addresses, users can use easy-to-read domain names.

## How DNS Works

1. User enters a website address.
2. The device asks a DNS server for the IP address.
3. The DNS server returns the correct IP address.
4. The device connects to the server.

## DNS Server

A DNS server is responsible for resolving domain names.

Common DNS servers:

- Google DNS: 8.8.8.8
- Cloudflare DNS: 1.1.1.1

## My Practice

I will check DNS settings on my Windows computer using:

ipconfig /all
