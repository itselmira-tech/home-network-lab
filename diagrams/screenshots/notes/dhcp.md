# DHCP (Dynamic Host Configuration Protocol)

## What is DHCP?

DHCP automatically assigns network settings to devices.

It provides:

- IP Address
- Subnet Mask
- Default Gateway
- DNS Server

## How DHCP Works

DHCP follows a process called DORA:

1. Discover
2. Offer
3. Request
4. Acknowledge

## DHCP Server

In most home networks, the router works as the DHCP server.
