# IP Addressing

## What is an IP Address?

An IP address is a unique address assigned to a device in a network.
It allows devices to communicate with each other.

## IPv4

IPv4 is a 32-bit addressing system.

Example:

192.168.1.10

An IPv4 address has four numbers separated by dots.

## Private IP Address

Private IP addresses are used inside local networks.

Common private IP ranges:

- 192.168.0.0 - 192.168.255.255
- 10.0.0.0 - 10.255.255.255
- 172.16.0.0 - 172.31.255.255

## Default Gateway

The default gateway is usually the router.
It connects the local network to the internet.
