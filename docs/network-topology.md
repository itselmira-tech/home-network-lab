# Component Description

## Router

The router provides:

- Internet access
- DHCP service
- Network routing
- Wireless connectivity


## Client Devices

Client devices connect to the router and receive network configuration automatically.

## Network Flow

1. Router connects to the internet.
2. Devices connect through Wi-Fi or Ethernet.
3. DHCP assigns IP addresses.
4. DNS resolves domain names.
5. Devices communicate inside the local network.
